# Швидкий старт / Quick Start Guide

## Для користувачів

### Крок 1: Завантажте програму
Завантажте файл `convert_csv_to_excel_v3.exe` з релізів GitHub.

### Крок 2: Встановіть контекстне меню (опціонально)

**Найпростіший спосіб:**
1. Помістіть `convert_csv_to_excel_v3.exe` у постійну папку (наприклад, `C:\Tools\CSVConverter\`)
2. Запустіть `install_context_menu.bat` **від імені адміністратора**
3. Готово! Тепер можна конвертувати файли правим кліком миші

### Крок 3: Використовуйте програму

**Спосіб 1: Через контекстне меню (якщо встановлено)**
1. Клікніть правою кнопкою на CSV файл
2. Виберіть "Конвертувати у Excel"
3. Програма автоматично виконає конвертацію

**Спосіб 2: Drag-and-Drop**
1. Відкрийте програму
2. Перетягніть CSV файли у вікно
3. Натисніть "🚀 Конвертувати у Excel"

**Спосіб 3: Через кнопку**
1. Відкрийте програму
2. Натисніть "📂 Вибрати файли"
3. Оберіть CSV файли
4. Натисніть "🚀 Конвертувати у Excel"

## Для розробників

### Встановлення залежностей
```bash
pip install -r requirements.txt
```

### Запуск з вихідного коду
```bash
python convert_csv_to_excel_v3.py
```

### Збірка .exe
```bash
pyinstaller convert_csv_to_excel_v3.spec
```

Готовий файл буде у папці `dist/`.
