## 🔨 Збірка виконуваного файлу

### Крок 1: Перевірка середовища
```bash
# Перевірка Python
python --version  # Має бути 3.8+

# Перевірка pip
pip --version
```

### Крок 2: Встановлення залежностей
```bash
pip install -r requirements.txt
```

### Крок 3: Збірка
```bash
pyinstaller convert_csv_to_excel_v3.spec
```

### Крок 4: Перевірка результату
Перевірте наявність файлу:
```
dist/CSVtoExcel.exe
```
