# Changelog / Історія змін

## [v3.1] - 2026-06-06

### Додано
- **Drag-and-Drop підтримка** - тепер можна перетягувати CSV файли безпосередньо у вікно програми
- **Інтеграція з контекстним меню Windows** - можливість конвертувати файли через правий клік у провіднику
- **Підтримка командного рядка** - автоматична конвертація при передачі файлів як аргументів
- Скрипти для встановлення контекстного меню:
  - `install_context_menu.bat` - найпростіший спосіб (Batch)
  - `install_context_menu.ps1` - PowerShell скрипт
  - `install_context_menu.reg` - ручне редагування реєстру
- Скрипти для видалення:
  - `uninstall_context_menu.bat`
  - `uninstall_context_menu.ps1`
  - `uninstall_context_menu.reg`
- `requirements.txt` для зручного встановлення залежностей
- `CONTEXT_MENU_INSTALLATION.md` - детальна інструкція зі встановлення
- `DEVELOPER_GUIDE.md` - керівництво для розробників
- Залежність `tkinterdnd2` для drag-and-drop функціональності

### Змінено
- Оновлено `convert_csv_to_excel_v3.py`:
  - Додано імпорт `tkinterdnd2`
  - Додано методи `on_drop()` та `process_command_line_args()`
  - Змінено `tk.Tk()` на `TkinterDnD.Tk()` для підтримки drag-and-drop
  - Зареєстровано drop target на listbox
- Оновлено `convert_csv_to_excel_v3.spec`:
  - Додано tkinterdnd2 до hiddenimports
  - Додано tkdnd файли до datas для коректної збірки
- Оновлено `README.md`:
  - Додано інформацію про нові можливості
  - Додано інструкції з встановлення залежностей
  - Оновлено структуру проєкту
- Оновлено `USER_MANUAL.md`:
  - Додано секцію про drag-and-drop
  - Додано секцію про контекстне меню
  - Оновлено покрокову інструкцію з трьома способами додавання файлів

### Технічні деталі
- При запуску через контекстне меню програма автоматично завантажує файл і починає конвертацію
- При перетягуванні файлів фільтруються тільки CSV файли
- Підтримка множинного вибору через drag-and-drop
- CSV файли, передані через командний рядок, автоматично додаються і конвертація стартує через 500мс

## [v3.0] - Попередня версія

### Можливості
- Масова конвертація CSV у Excel
- Очищення даних від HTML-ентітей та зайвих символів

- Групування (ua) та (pl) стовпців
- Форматування Excel (жирні заголовки, автоширина)
- Розбиття великих файлів
- Фоновий потік для конвертації
- Прогрес-бар та журнал процесу
