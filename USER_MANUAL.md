## Що робить програма

**CSV → Excel Конвертер** перетворює файли формату `.csv` у файли формату `.xlsx` (Microsoft Excel). При цьому програма:

- читає дані з CSV-файлу,
- за бажанням очищає їх від зайвих символів (HTML-теги, подвійні пробіли тощо),
- перейменовує технічні назви стовпців на зрозумілі (за словником),
- групує україномовні варіанти стовпців `(ua)` поруч із базовими,
- форматує результуючу таблицю (жирний заголовок, автоширина колонок),
- зберігає готовий `.xlsx`-файл поруч із вихідним `.csv`.

---

## Системні вимоги

- **Операційна система:** Windows 7 / 10 / 11 (64-bit)
- **Місце на диску:** ~50 МБ для виконуваного файлу
- **Оперативна пам'ять:** від 256 МБ (для великих файлів — більше)
- Встановлення Python або будь-яких інших програм **не потрібне** — `.exe` працює автономно.

---

## Запуск програми

1. Двічі клацніть на файл `convert_csv_to_excel_v3.exe`.
2. Якщо Windows показує попередження «Невідомий видавець» — натисніть **«Все рівно запустити»**.
3. Головне вікно програми відкриється автоматично.

## Покрокова інструкція

### 1. Вибрати файли

Натисніть кнопку **📂 Вибрати файли**.  
У вікні файлового провідника оберіть один або декілька `.csv`-файлів (для множинного вибору утримуйте `Ctrl` або `Shift`).  
Натисніть **Відкрити** — файли з'являться у списку.

### 2. Налаштувати параметри

**Форматування:**
- **Очищений** *(рекомендовано)* — видаляє HTML-ентіті, зайві пробіли, керуючі символи; замінює коми всередині значень на крапку з комою.
- **Оригінальний** — зберігає дані без змін, лише конвертує формат.

**Видалити CSV після конвертації:**  
Поставте галочку, якщо хочете, щоб програма автоматично видаляла вихідні CSV-файли після успішного збереження xlsx.  
> ⚠️ Видалення є незворотним. Впевніться, що результат вас влаштовує, або тримайте цю опцію вимкненою.

**Групувати (ua)-стовпці поруч із базовими:**  
Якщо у файлі є стовпці з числовим ID та їх україномовні аналоги (наприклад, `Назва;12345` і `Назва (ua);12345`), програма поставить їх поруч для зручного читання.

### 3. Запустити конвертацію

Натисніть **🚀 Конвертувати у Excel**.  

- Прогрес-бар відображатиме відсоток виконання.
- Журнал показуватиме деталі по кожному файлу.
- Після завершення з'явиться повідомлення з підсумками.

### 4. Знайти результат

Готові `.xlsx`-файли збережені **у тій самій папці**, де знаходились вихідні CSV-файли, з тією ж назвою але розширенням `.xlsx`.

**Приклад:**
```
C:\Звіти\products.csv  →  C:\Звіти\products.xlsx
```

## Налаштування

### Що відбувається з великими файлами?

Якщо CSV-файл перевищує **200 МБ**, програма попередить вас та запитає підтвердження перед початком обробки — конвертація великих файлів може займати тривалий час.

Якщо кількість рядків у файлі перевищує **1 048 575** (ліміт аркуша Excel), файл автоматично розбивається на частини:
```
products.csv  →  products_part1.xlsx
              →  products_part2.xlsx
              →  ...
```

### Захист від перезапису

Якщо файл `products.xlsx` вже існує, програма **не перезапише** його, а створить `products_1.xlsx` (або `_2`, `_3` і т. д.).

## Словник заголовків

Програма може автоматично перейменовувати технічні назви стовпців на зрозумілі.

### Відкрити словник

Натисніть **⚙** (іконка шестерні) у правому куті заголовка програми.

### Структура словника

Кожен рядок — це пара:

```
Оригінальна назва  →  Нова назва
```

**Приклади за замовчуванням:**

| Оригінальна назва | Нова назва |
|---|---|
| `goods_article` | `Артикул` |
| `goods_title` | `Название` |
| `goods_title_uk` | `Назва (ua)` |
| `goods_status` | `Статус` |

### Редагування словника

- **Додати слово** — додає нову пару перейменування.
- **Видалити останній** — видаляє останній доданий рядок.
- **Зберегти** — зберігає зміни у файл `header_map.json` (поруч з програмою); словник завантажується автоматично при наступному запуску.
- **Скасувати** — закриває вікно без збереження.

> Якщо стовпець з такою назвою не знайдений у CSV — перейменування просто ігнорується.

## Журнал процесу

Журнал у нижній частині вікна відображає:

| Символ | Значення |
|---|---|
| 📂 | Читання файлу |
| ✅ | Файл успішно збережено |
| 🗑️ | CSV-файл видалено |
| ❌ | Помилка при обробці файлу |

Після конвертації всього пакету відображається підсумок:
```
Завершено! Успішно: 5, з помилкою: 0
```

## Типові ситуації та вирішення

| Ситуація | Що робити |
|---|---|
| Файл `.csv` не відображається у діалозі | Переконайтесь, що вибрано тип «CSV files» або «All files» |
| Стовпці не перейменовуються | Перевірте словник — назва стовпця має збігатися точно |
| Дані у Excel відображаються некоректно | Спробуйте режим «Очищений» — він нормалізує спецсимволи |
| Файл `.xlsx` не з'являється | Перегляньте журнал на наявність рядка ❌ — там буде опис помилки |
| Програма не запускається | Переконайтесь, що файл не заблокований антивірусом; спробуйте «Запустити від імені адміністратора» |
| Конвертація дуже повільна | Великі файли (100+ МБ) обробляються довше; зачекайте або обробляйте файли частинами |
