# Інструкція по збірці професійного .exe файлу

## Підготовка середовища

### 1. Встановлення Python
Переконайтеся, що у вас встановлено Python 3.8 або новіше:
```bash
python --version
```

### 2. Встановлення залежностей
```bash
pip install -r requirements.txt
```

## Збірка виконуваного файлу

### Використання PyInstaller

Проект вже містить налаштований файл `convert_csv_to_excel_v3.spec` з усіма необхідними параметрами для професійної збірки:

```bash
pyinstaller convert_csv_to_excel_v3.spec
```

### Що включає збірка

✅ **Логотип програми** - файл `logo.ico` автоматично вбудовується як іконка .exe  
✅ **Бібліотека tkinterdnd2** - підтримка drag-and-drop включена  
✅ **Усі залежності** - pandas, openpyxl, tkinter упаковані в один файл  
✅ **Режим без консолі** - програма запускається без cmd вікна  
✅ **UPX стиснення** - зменшений розмір файлу  

### Результат збірки

Готовий виконуваний файл з'явиться в:
```
dist/CSVtoExcel.exe
```

## Розповсюдження

### Файли для розповсюдження

Для повноцінної роботи програми користувачу потрібні:

1. **CSVtoExcel.exe** - основний виконуваний файл
2. **header_map.json** (опціонально) - попередньо налаштований словник заголовків

### Файли контекстного меню (опціонально)

Якщо ви хочете надати користувачам можливість інтеграції з контекстним меню Windows:

- `install_context_menu.bat` - встановлення контекстного меню
- `uninstall_context_menu.bat` - видалення контекстного меню
- `CONTEXT_MENU_INSTALLATION.md` - інструкція по встановленню

### Документація для користувачів (опціонально)

- `USER_MANUAL.md` - детальна інструкція користувача
- `QUICK_START.md` - швидкий старт

## Тестування зібраного .exe

1. Перейдіть до папки `dist/`
2. Запустіть `CSVtoExcel.exe`
3. Перевірте:
   - ✅ Іконка програми відображається коректно
   - ✅ Вікно відкривається без помилок
   - ✅ Drag-and-drop працює
   - ✅ Конвертація файлів виконується успішно
   - ✅ Словник заголовків зберігається і завантажується

## Поширені проблеми

### Програма не запускається
- Переконайтеся, що `logo.ico` присутній під час збірки
- Перевірте, що всі залежності встановлені

### Помилка "tkdnd not found"
- Файл `.spec` автоматично включає tkdnd бібліотеку
- Переконайтеся, що tkinterdnd2 встановлено: `pip install tkinterdnd2`

### Великий розмір .exe файлу
- Це нормально для standalone застосунку з усіма залежностями
- Розмір ~50-80 МБ є типовим для програм з pandas та openpyxl

## Налаштування збірки

Якщо потрібно змінити параметри збірки, відредагуйте `convert_csv_to_excel_v3.spec`:

```python
# Назва виконуваного файлу
name='convert_csv_to_excel_v3',

# Іконка
icon=['logo.ico'],

# Режим консолі (False = без консолі)
console=False,

# UPX стиснення (True = включено)
upx=True,
```

## Створення інсталятора (опціонально)

Для ще більш професійного вигляду можна створити інсталятор за допомогою:
- **Inno Setup** (безкоштовний) - https://jrsoftware.org/isinfo.php
- **NSIS** (безкоштовний) - https://nsis.sourceforge.io/
- **Advanced Installer** (комерційний)

Інсталятор може:
- Автоматично встановлювати програму в Program Files
- Створювати ярлик на робочому столі
- Додавати програму до меню Пуск
- Автоматично реєструвати контекстне меню

## Версіонування
