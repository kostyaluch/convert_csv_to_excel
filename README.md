# CSV → Excel Конвертер v3

Зручний десктопний застосунок для швидкого масового перетворення файлів формату CSV у формат Excel (`.xlsx`). Програма не просто змінює розширення — вона автоматично форматує таблицю для зручного читання, очищає дані від «сміття» та підтримує гнучке перейменування заголовків стовпців.

---

## Можливості

- **Масова конвертація** — обирайте одразу кілька CSV-файлів і конвертуйте їх одним натисканням.
- **Очищення даних** — видалення HTML-ентітей, зайвих пробілів, керуючих символів та нормалізація пунктуації.
- **Форматування Excel** — жирні заголовки, автоматична ширина стовпців, перенос тексту у клітинках.
- **Словник заголовків** — перейменовує технічні назви колонок на читабельні (редагується прямо в інтерфейсі та зберігається між сесіями).
- **Групування (ua)-колонок** — автоматично переставляє україномовні варіанти стовпців поруч із базовими.
- **Розбиття великих файлів** — якщо рядків більше за ліміт Excel (1 048 576), файл автоматично ділиться на частини `_part1.xlsx`, `_part2.xlsx` тощо.
- **Захист від перезапису** — якщо файл з такою назвою вже існує, новий файл отримає суфікс `_1`, `_2` і т. д.
- **Видалення вихідних CSV** — опціональне автовидалення CSV-файлів після успішної конвертації.
- **Попередження про великі файли** — при спробі конвертувати файл понад 200 МБ програма запитає підтвердження.
- **Журнал процесу** — детальний лог у вікні програми з відображенням прогресу у відсотках.

## Технічний стек

| Компонент | Версія |
|---|---|
| Python | 3.x |
| pandas | для читання CSV |
| openpyxl | для запису та форматування xlsx |
| tkinter | графічний інтерфейс |
| PyInstaller | збірка у `.exe` |

---

## Збірка .exe

```bash
pyinstaller convert_csv_to_excel_v3.spec
```

Готовий виконуваний файл з'явиться у папці `dist/`.
