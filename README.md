# CSV → Excel Конвертер v3

Зручний десктопний застосунок для швидкого масового перетворення файлів формату CSV у формат Excel (`.xlsx`). Програма не просто змінює розширення — вона автоматично форматує таблицю для зручного читання, очищає дані від «сміття» та підтримує гнучке перейменування заголовків стовпців.

---

## Можливості

### Конвертація файлів

- **Масова конвертація** — обирайте одразу кілька CSV-файлів і конвертуйте їх одним натисканням кнопки.
- **Drag-and-Drop** — перетягніть CSV-файли безпосередньо у вікно програми для швидкого додавання.
- **Контекстне меню Windows** — натисніть правою кнопкою миші на CSV файл у провіднику та виберіть "Конвертувати у Excel". Детальніше в [CONTEXT_MENU_INSTALLATION.md](CONTEXT_MENU_INSTALLATION.md).
- **Командний рядок** — передавайте файли як аргументи для автоматичної конвертації.
- **UTF-8 кодування** — CSV-файли зчитуються у кодуванні UTF-8 з комою як роздільником.
- **Фоновий потік** — конвертація виконується у окремому потоці; інтерфейс залишається активним і відповідним під час обробки.
- **Прогрес-бар** — відображає відсоток виконання у реальному часі.
- **Журнал процесу** — детальний лог у вікні програми: кількість рядків і стовпців, час обробки кожного файлу, підсумкові результати.

### Очищення даних (режим «Очищений»)

- **Видалення HTML-ентітей** — виконується до 5 ітерацій `html.unescape`, щоб повністю розкодувати вкладені ентіті.
- **Нормалізація ком** — коми всередині значень замінюються на крапку з комою (`;`), що запобігає помилковому парсингу.
- **Видалення керуючих символів** — прибираються `\r\n`, `\n`, `\r`, `\t`, `\f`, `\v`.
- **Скорочення пробілів** — кілька підряд пробілів зводяться до одного, рядок обрізається.
- **Обробка екранованих апострофів** — послідовність `\'` замінюється на `'`.
- Числові та цілі значення **не змінюються** — очищення застосовується лише до рядкових клітинок.

### Форматування Excel

- **Жирні заголовки** — перший рядок виділяється жирним шрифтом.
- **Висота рядка заголовка** — фіксована висота 50 pt для кращої читабельності.
- **Автоматична ширина стовпців** — ширина підбирається за найдовшим значенням у стовпці (мінімум 15, максимум 50 знаків).
- **Перенос тексту** — у клітинках увімкнено `wrap_text`; вирівнювання — по верхньому краю ліворуч.

### Словник заголовків

- **Перейменування колонок** — технічні назви стовпців автоматично замінюються на зрозумілі відповідно до словника.
- **Вбудований словник за замовчуванням** — містить 8 готових відповідностей:

  | Оригінальна назва | Нова назва |
  |---|---|
  | `goods_article` | `Артикул` |
  | `goods_category_title` | `Назва категорії` |
  | `goods_docket` | `Краткое описание` |
  | `goods_docket_uk` | `Короткий опис (ua)` |
  | `goods_status` | `Статус` |
  | `goods_state` | `Стан` |
  | `goods_title` | `Название` |
  | `goods_title_uk` | `Назва (ua)` |

- **Редагований через інтерфейс** — відкривається кнопкою ⚙ у заголовку програми; підтримує додавання та видалення рядків.
- **Збереження між сесіями** — словник зберігається у файл `header_map.json` поруч із програмою і автоматично завантажується при наступному запуску.
- Якщо стовпець із зазначеною назвою відсутній у CSV — перейменування ігнорується без помилок.

### Групування (ua)-стовпців

- Автоматично переставляє україномовний варіант стовпця (`X (ua)`) одразу після базового (`X`).
- Застосовується **лише до атрибутних стовпців**, що містять числовий ID (наприклад, `Назва;20775` і `Назва (ua);20775`). Сервісні заголовки без цифр (як `Назва (ua)`) залишаються на місці.
- Журнал показує кількість знайдених та переміщених пар.
- Можна вимкнути через відповідну галочку в налаштуваннях.

### Захист і надійність

- **Розбиття великих файлів** — якщо рядків більше за ліміт аркуша Excel (1 048 575 рядків даних), файл автоматично ділиться на `_part1.xlsx`, `_part2.xlsx` тощо.
- **Захист від перезапису** — якщо файл із такою назвою вже існує, новий отримає суфікс `_1`, `_2` і т. д.
- **Попередження про великі файли** — при спробі конвертувати файл понад 200 МБ програма запитає підтвердження перед стартом.
- **Видалення вихідних CSV** — опціональне автовидалення CSV-файлів після успішної конвертації (вмикається галочкою).

## Технічний стек

| Компонент | Призначення |
|---|---|
| Python 3.x | мова програмування |
| pandas | зчитування CSV |
| openpyxl | запис та форматування `.xlsx` |
| tkinter | графічний інтерфейс |
| tkinterdnd2 | підтримка drag-and-drop |
| threading | фонова обробка без заморожування UI |
| PyInstaller | збірка у автономний `.exe` |

---

## Встановлення залежностей

Перед збіркою програми встановіть необхідні бібліотеки:

```bash
pip install -r requirements.txt
```

Або встановіть вручну:

```bash
pip install pandas openpyxl tkinterdnd2 pyinstaller
```

## Збірка .exe

### Швидкий старт

```bash
pyinstaller convert_csv_to_excel_v3.spec
```

Готовий виконуваний файл з'явиться у папці `dist/CSVtoExcel.exe`.

- ✅ **Метаданими версії** у властивостях .exe
- ✅ **UPX стисненням** для зменшення розміру
- ✅ **Режимом без консолі** - чисто GUI застосунок

## Використання контекстного меню

Після встановлення контекстного меню:
1. Натисніть правою кнопкою миші на будь-який CSV файл у провіднику Windows
2. Виберіть **"Конвертувати у Excel"**
3. Програма автоматично запуститься та сконвертує файл

Детальна інструкція з встановлення доступна в файлі [CONTEXT_MENU_INSTALLATION.md](CONTEXT_MENU_INSTALLATION.md).
