# Інструкція з встановлення контекстного меню

## Що таке контекстне меню?

Контекстне меню дозволяє натиснути правою кнопкою миші на CSV файл у провіднику Windows та вибрати опцію "Конвертувати у Excel". Програма автоматично запуститься та конвертує вибраний файл.

## Встановлення контекстного меню

### Спосіб 1: Batch скрипт (найпростіший)

1. Розпакуйте програму у будь-яку папку на вашому комп'ютері
2. Натисніть правою кнопкою миші на файл `install_context_menu.bat`
3. Виберіть **"Запустити від імені адміністратора"** або **"Run as administrator"**
4. Натисніть будь-яку клавішу після завершення

### Спосіб 2: PowerShell скрипт

1. Розпакуйте програму у будь-яку папку на вашому комп'ютері
2. Натисніть правою кнопкою миші на файл `install_context_menu.ps1`
3. Виберіть **"Виконати з PowerShell"** або **"Run with PowerShell"**
4. Підтвердіть дію адміністратора (якщо запитає система)
5. Натисніть будь-яку клавішу після завершення

**Примітка:** Якщо виникає помилка політики виконання, виконайте наступну команду в PowerShell від імені адміністратора:
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

### Спосіб 3: Ручне редагування реєстру

1. Відкрийте файл `install_context_menu.reg` у текстовому редакторі
2. Замініть `%INSTALL_PATH%` на повний шлях до папки з програмою
   
   Наприклад, якщо програма знаходиться в `C:\Tools\CSV_Converter\`, то рядок:
   ```
   "Icon"="%INSTALL_PATH%\\convert_csv_to_excel_v3.exe,0"
   ```
   повинен стати:
   ```
   "Icon"="C:\\Tools\\CSV_Converter\\convert_csv_to_excel_v3.exe,0"
   ```
   
3. Збережіть файл
4. Двічі клацніть на `install_context_menu.reg`
5. Підтвердіть додавання запису до реєстру

## Використання контекстного меню

1. Знайдіть CSV файл у провіднику Windows
2. Натисніть на нього правою кнопкою миші
3. Виберіть **"Конвертувати у Excel"**
4. Програма автоматично запуститься та розпочне конвертацію

## Видалення контекстного меню

### Спосіб 1: Batch скрипт (найпростіший)

1. Натисніть правою кнопкою миші на файл `uninstall_context_menu.bat`
2. Виберіть **"Запустити від імені адміністратора"** або **"Run as administrator"**
3. Натисніть будь-яку клавішу після завершення

### Спосіб 2: PowerShell скрипт

1. Натисніть правою кнопкою миші на файл `uninstall_context_menu.ps1`
2. Виберіть **"Виконати з PowerShell"** або **"Run with PowerShell"**
3. Підтвердіть дію адміністратора (якщо запитає система)
4. Натисніть будь-яку клавішу після завершення

### Спосіб 3: Ручне редагування реєстру

1. Двічі клацніть на `uninstall_context_menu.reg`
2. Підтвердіть видалення запису з реєстру

## Drag-and-Drop (перетягування файлів)

Тепер ви також можете:
- Перетягнути один або кілька CSV файлів безпосередньо у вікно програми
- Файли автоматично додадуться до списку для конвертації

## Підтримка командного рядка

Програма підтримує передачу файлів через командний рядок. Це дозволяє:
- Запускати конвертацію з bat-скриптів
- Інтегрувати з іншими програмами
- Використовувати контекстне меню Windows

Приклад:
```batch
convert_csv_to_excel_v3.exe "C:\data\file1.csv" "C:\data\file2.csv"
```

При запуску з аргументами командного рядка, програма автоматично завантажить вказані файли та розпочне конвертацію.
